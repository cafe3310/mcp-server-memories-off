# 提示词和描述的术语对应

`meta file` - 描述文件
`library` - 知识库
`name` (of library) - 知识库名称
`reason` (of tool calling) - (调用的)目的

`entity` - 知识实体, 实体，文档
`type` (of entity) - (知识实体)类型
`alias` (of entity) - (知识实体)别名
`content` (of entity) - (知识实体)内容

`glob pattern` - glob 模式
`TOC` - 目录
`section` - 章节
`exact whole line or multiple lines of content` - 精确的整行或多行内容
